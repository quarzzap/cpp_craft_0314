Неделя 3. Пояснение к шаблону проекта. 
======================================

Важно:
1. Данный шаблонный проект используется совместно с шаблонным проектом 2-ой недели, на основании которого вы создавали ваше решение. Я обновил проект в ihar_marozau/2, теперь он содержит полный код библиотеки. Можете его использовать.

2. Появился файндер для статической библиотеки под названием binary_reader: _cmake_scripts/Findbinary_reader.cmake. Он нужен для того, чтобы подлючить вашу статическую библиотеку из решения задач 2-ой недели.

3. Так же как и с бибилиотекой boost, если вы хотите подключить binary_reader к проекту, то CMakeLists.txt примет следующей вид:

find_package( binary_reader ${SEARCH_PARAMETERS} COMPONENTS
	binary_reader
)

compile_project( ${module_name} "*.cpp" "*.h" BINARY binaries binary_reader )


!!!Важно!!! Перед тем как подлючать библиотеку не забудьте скопилировать проект ее содержащий. Данный файндер настроен на имя библиотеки binary_reader.
